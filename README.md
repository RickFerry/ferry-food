## 💻 Sobre o projeto

Ferryfood é uma aplicação fast-food de microsserviços, onde se realiza pedidos(requests) e se faz pagamentos(payments), com end points que oferecem CRUD basicos para ambos os recursos.

Aplicação sem implementação mobile ou front-end, havendo unicamente a disponibilidade da API.

---

## ⚙️ Funcionalidades

- [x] CRUD de pagamentos(payments);
- [x] CRUD de pedidos(requests);
